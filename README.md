# 💎 Simon Game

A modern and visually stunning memory sequence game inspired by the classic Simon Game. Test your memory by repeating an ever-growing sequence of glowing gems while enjoying immersive sound effects, smooth animations, and a futuristic neon interface.

## 🎯 How to Play

1. Click **Start** to begin the game.
2. Watch the sequence of glowing gems carefully.
3. Repeat the exact sequence by clicking the gems.
4. Each successful round adds a new gem to the sequence.
5. Make a mistake and the game ends.
6. Try to beat your highest score!

## 🎨 Game Features Breakdown

### 💎 Gem Board

Four interactive gems:

* 🔴 Red
* 🔵 Blue
* 🟢 Green
* 🟡 Yellow

Each gem has:

* Unique glow effect
* Individual sound frequency
* Interactive animations

### 🔊 Audio System

Built using the **Web Audio API** to generate sounds dynamically without external audio files.

### 📈 Difficulty Scaling

The game becomes faster as you progress:

| Round | Speed       |
| ----- | ----------- |
| 1 - 5 | Normal      |
| 6 - 9 | Faster      |
| 10+   | Expert Mode |
